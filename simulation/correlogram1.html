Instructions for Spectral Estimation using Correlogram

  • Step 1: Click on "Generate Input Signal" to generate the signal.
  • Step 2: Select the base signal from the dropdown menu, and enter the sampling frequency (Hz), length of the signal, and the input signal frequency value (Hz) in the parameters section.
  • Step 3: Add AWGN noise by entering the SNR value (in dB) in the AWGN noise section.
  • Step 4: Click the "Simulate" button to run the simulation.
  • Step 5: Reset the simulator by clicking the "Reset Simulator" button.
